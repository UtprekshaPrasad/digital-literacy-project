# Digital Literacy Project
## Details
* Name- Utpreksha Prasad
* Reg. no- 25BCE10108
## Overview
This digital literacy project aims to empower students by teaching essential skills of digital world by enabling them to use digital devices, the internet and online services safely and effectively. This project is divided into 5 modules with each containing its own different tasks.
## Module 1:
* It involves creating a digital literacy awareness infographic through a free presentation degign tool like Canva, Prezi etc. by covering topics such as what is digital literacy, useful digital tools for students, safe internet practices etc.

## Module 2:
* It includes creating a student digital portfolio through atleast 3 academic platforms like GitHub, Linkedin, Researchgate, Kaggle etc. It helps the student to reflect on their learning journey and set up a searchable portfolio.

## Module 3:
* It contains exploring coding platforms like HackerRank, CodeChef etc. to improvise a student's coding and problem solving skills through challenges. It also explores collaborative platform by creating google form of 'digital literacy awareness quiz' which can be shared through anyone via a link.

## Module 4:
* It involves drafting two professional emails which helps in understanding effective digital communication and the ettiquettes present. It also contains creating a social media checklist for understanding the responsible use of social media.

## Module 5:
* It includes having a case study on any of the cybercrimes like phishing, cyberbullying, malware etc. to create awareness and also creating a prevention checklist based on staying safe online that is especially relevant to college students.
